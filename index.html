Features

BUILT FOR
REAL workflows

AgentFlow doesn't just chat — it reads your portal context, understands your staff roles, and takes actions within your existing system.


Fully configurable via API key. Theme-matched to your brand. Logs every interaction.

Context-Aware Agent
The widget reads the DOM of your portal — it knows which page the user is on, what records are visible, and who is logged in.
Real-Time Audit Trail
Every AI interaction is logged with timestamp, user identity, action taken, and the reasoning chain used to reach the answer.
Branded Themes
Choose from built-in color themes or pass custom hex values. The widget adapts to your brand automatically with zero CSS work.
Conversational Memory
Maintains context across a session so staff can ask follow-ups naturally, just like talking to a knowledgeable colleague.
File & Document Support
Staff can attach PDFs, images, and documents directly in chat. The agent extracts and reasons over the file content instantly.
API-First Architecture
Built on a clean REST backend. Configure system prompts, personas, and tool access via API key — no dashboard required.
How it works

DEPLOY IN
minutes

1
Get your API key
Sign up and receive a unique API key for your organization. Configure your agent's persona and knowledge base in the dashboard.
2
Drop in one script tag
Add a single <script> tag to your portal. Pass your API key and preferred theme — AgentFlow handles the rest.
3
Agent goes live instantly
A floating AI widget appears on your portal. It reads the page context, identifies the logged-in user, and starts assisting immediately.
4
Monitor via audit dashboard
Every conversation, decision, and action is captured and viewable in your audit trail with timestamps and user attribution.
embed.html HTML
<!-- Step 1: Add to your portal's </body> -->
 
<script
  src="https://cdn.agentflow.io/widget.js"
  data-api-key="af_live_yourkey123"
  data-theme="blue"
  defer
></script>
 
<!-- That's it. The agent is now live. -->
 
// Optional: configure via JS
window.AgentFlow?.configure({
  persona: "Claims Assistant",
  greeting: "Hi! How can I help?",
  theme: { primary: "#1a3a6b" }
});
